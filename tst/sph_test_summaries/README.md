# Spherical-Shell Validation Summaries

This directory collects human-readable reports from local/cluster validation
runs. Generated logs, CSVs, plots, binary outputs, and helper scripts remain in
ignored validation directories and are not committed here.

Included reports:

- `cluster_validation_summary.md` - original hydro CPU/GPU validation and vibe checks.
- `mhd_output_validation_summary.md` - spherical-shell MHD monopole residual and native binary output check.
- `mhd_vibe_validation_summary.md` - MHD native-bin vibe checks.
- `aw_damping_scan_summary.md` - first Alfvén-wave damping scan.
- `aw_damping_scan_replot_v2_report.md` - audit/replot of AW damping diagnostics.
- `wave_action_conservation_summary.md` - driven-wave setup notes and initial baseline.
- `wave_action_conservation_baseline_debug_summary.md` - monopole transverse-drift debug.
- `monopole_hlld_interface_debug_summary.md` - HLLD radial-interface pathology investigation.
- `monopole_hlld_interface_debug_hlld_comparison_notes.md` - AthenaK/Athena++ HLLD comparison notes.
- `monopole_lhlld_minimal_summary.md` - LHLLD minimal port and Parker HLLD follow-up.
